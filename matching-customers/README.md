# matching-customers

Read a list of customers, and output the customers living within 100km to Dublin.

## setup:

* Install Node (Version 4.0 or above)
* Install all dependencies:
  `npm install`
* Install Mocha test runner
  `npm install -g mocha`
* To run the program
   `node matching-customers.js`
* To run the tests: 
  `npm test`
